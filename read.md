## Backend

1. Criar ficheiro package.json
   - npm init -y
2. Instalar express
   - npm install express
3. Instalar nodemon
   - npm install nodemon
4. Criar script
   - dev: nodemon server.js
5. Iniciar servidor
   - npm run dev
6. Instalar dotenv
   - Criar ficheiro .env
     - PORT=4000
7. Criar conta na plataforma ATLAS
   - Criar cluster, utilizador e senha
   - Copiar codigo (driver) para inserir na aplicação e poder conectar
8. Instalar mongoose
   - npm install mongoose
